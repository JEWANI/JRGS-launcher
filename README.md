## ✨ 주요 기능

- ROM 폴더 자동 스캔 및 플랫폼 자동 분류
- 게임 그리드 뷰 (이름만 / 작은 아이콘 / 큰 아이콘)
- ScreenScraper API 메타데이터 자동 검색 (커버아트, 장르, 제작사 등)
- 에뮬레이터 등록/편집 및 게임별 에뮬레이터 개별 지정
- 유튜브 인앱 재생 (재와니 채널 우선 자동 검색)
- 플레이 타임 기록 및 통계
- 스크린샷 F12 캡처
- 외부 녹화 프로그램 연동 (반디캠, OBS 등)
- MORT 실시간 번역기 연동
- 다크 / 레트로 베이지 / 그린 레트로 테마 및 커스텀 테마
- 백업 / 복원 기능

## 🕹️ 주요 단축키

| 기능 | 단축키 |
|------|--------|
| 현재 탭 ROM 스캔 | F5 |
| 전체 ROM 스캔 | Shift+F5 |
| ROM 개별 추가 | Ctrl+O |
| 게임 실행 | Enter |
| 즐겨찾기 추가/제거 | Ctrl+D |
| 스크린샷 | F12 |
| 녹화 시작/중지 | Ctrl+R |
| 아이콘 크기 조절 | Ctrl+Shift+휠 |
| 전체화면 | Alt+Enter |
